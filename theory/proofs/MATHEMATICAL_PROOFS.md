### Theorem 1: Uniqueness of E₈ (Viazovska 2016)

**Theorem:** The E₈ lattice achieves the densest sphere packing in ℝ⁸, and this solution is unique.

**Proof Sketch:**
Viazovska constructed an auxiliary function f: ℝ⁸ → ℝ satisfying:
1. f(0) = f̂(0) > 0 (where f̂ is Fourier transform)
2. f(x) ≤ 0 for |x| ≥ √2
3. f̂(t) ≥ 0 for all t

Such a function f is called a "magic function" for the sphere packing bound.

The existence of this function proves that no packing can exceed the E₈ density.
The E₈ lattice achieves this bound exactly.

**Reference:** Viazovska, M. (2017). "The sphere packing problem in dimension 8." Annals of Mathematics, 185(3), 991-1015.

**Theorem:** E₈ is self-dual: E₈* = E₈.

**Proof:**

### Theorem 5: Action Minimization

**Theorem:** The H₄-preserving projection Π_H₄ is the unique minimum of:
```
S[Π] = ∫_{E₈} (|∇Π|² + λ|Π - Π_H₄|²) d⁸x
```

**Proof:**
The Euler-Lagrange equation is:
```
-∇²Π + λ(Π - Π_H₄) = 0
```

For Π = Π_H₄:
- ∇²Π_H₄ = 0 (Π_H₄ is linear, hence harmonic)
- |Π_H₄ - Π_H₄|² = 0

Therefore Π_H₄ satisfies the E-L equation trivially.

For uniqueness: The Hessian of S at Π_H₄ is positive definite:
```
δ²S/δΠ² = -∇² + λ > 0
```
since λ > 0 and -∇² ≥ 0.

Therefore Π_H₄ is a strict minimum. ∎
