# GSM Formula Reference

The canonical formula reference has been consolidated to a single location:

**[../FORMULAS.md](../FORMULAS.md)**

This includes all 58 constant derivations, master equations, the allowed exponent set, and quick verification code snippets.

## Verification Scripts

Each script in this directory independently re-derives a subset of constants from φ, π, and E₈ structural numbers. See [DERIVATIONS_INDEX.md](DERIVATIONS_INDEX.md) for the full index.

## Validation Tiers

| Tier | Gate | Description |
|------|------|-------------|
| A | < 0.01% | Ultra-precise constants (α⁻¹, mμ/me, mp/me). Sigma gated on percent only — experimental uncertainties are sub-ppb. |
| B | < 1%, < 5σ | Well-measured constants (gauge couplings, mass ratios, CKM/PMNS, boson masses). |
| C | < 2%, < 5σ | Percent-level constants (cosmological parameters, neutrino masses). |
| Q | Informational | Light quark absolute masses — pole-to-MS-bar running shifts ~15-30%. Mass *ratios* are the physical predictions. |
| P | Prediction | Untested predictions (CHSH bound, Δm² splittings, tensor-to-scalar ratio). |
